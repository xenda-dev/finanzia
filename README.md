# 💰 FinanzIA

**FinanzIA** es una aplicación SaaS de finanzas personales diseñada para ayudar a las personas a tomar el control de su dinero de manera simple, visual e inteligente. La plataforma permite gestionar ingresos, gastos, presupuestos, deudas, metas de ahorro e inversiones, de forma intuitiva y segura, con sincronización en la nube y acceso multiplataforma mediante tecnología PWA.

## 🚀 Características principales

- 📊 **Dashboard financiero** con indicadores clave.
- 💳 **Gestión de cuentas y transacciones** (ingresos, gastos y transferencias).
- 📅 **Presupuestos y control de gastos**.
- 🎯 **Metas de ahorro** y seguimiento de progreso.
- 💸 **Gestión de deudas y pagos programados**.
- 📈 **Análisis financiero** con reportes visuales.
- 🌍 **Soporte multidivisa**.
- 🌐 **Multiusuario con sincronización en la nube** mediante Supabase.
- 🔐 **Autenticación segura** con correo/contraseña y biometría (WebAuthn).
- 📱 **Aplicación PWA instalable** en Android, iOS y escritorio.
- 🌙 **Modo oscuro y claro**.
- 🤖 **Asistente financiero con IA**.

## 🛠️ Tecnologías utilizadas

- **Frontend:** HTML, CSS y JavaScript puro (sin frameworks).
- **Backend & Autenticación:** Supabase.
- **Base de datos:** PostgreSQL.
- **Despliegue:** Vercel.
- **Formato de aplicación:** Progressive Web App (PWA).

## 🌐 Acceso a la aplicación

👉 https://finanzia.xenda.co

## 📦 Instalación como PWA

1. Accede a la URL desde tu navegador.
2. Selecciona **“Instalar aplicación”** o **“Añadir a la pantalla de inicio”**.
3. Inicia sesión y comienza a gestionar tus finanzas.

## 🔒 Seguridad

- Autenticación segura mediante Supabase.
- Soporte para biometría (huella o reconocimiento facial).
- Sin almacenamiento de información sensible en el cliente sin cifrado.
- Sincronización segura en la nube.
